**Safe paths:**

- Focus on repos under `/home/ubuntu/fleet-knowledge/` and other known SuperInstance paths.

3. **Evaluate each README against criteria:**
   - **Essential Sections:** Description, Installation, Usage, API/Examples.
   - **Clarity:** Language is clear, not overly technical or vague.
   - **Recency:** Last updated within the last 6 months.
   - **Completeness:** Links to related repos, contribution guidelines.

4. **Score and rank (1=worst, 10=best):**
   - **1 point** per missing essential section.
   - **1 point** if unclear or overly jargon-heavy.
   - **1 point** if outdated (>6 months).
   - **1 point** if lacks links to related projects.
