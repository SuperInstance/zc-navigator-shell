3. **Structure (0-3 points):** Has at least one header (e.g., `# Title`) = 1, has sections = +1, has list/code = +1.
4. **Clarity (0-2 points):** Clearly states purpose (1 point), provides usage example (1 point).

2. Score each using criteria above.
3. Sort by total score (ascending).
