**Execution plan:**
1. Scan for README.md files in fleet directories.
2. For each, extract metadata: repo name, README size, last modified.
3. Score each README on a simple rubric (0–5):

   - 3: Incomplete (missing sections like install, usage)
   - 4: Adequate but could be improved

4. Sort by score ascending, pick top 10.
5. Output list with repo path, score, and brief reason.
