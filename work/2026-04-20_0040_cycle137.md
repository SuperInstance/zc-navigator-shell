### Step 3 — Optimize Within Channel (P2)
**Execution plan:**
1. Locate SuperInstance repos: `find /home -type d -name ".git" -path "*SuperInstance*" 2>/dev/null | head -20`
2. For each repo, find README: `find <repo_root> -maxdepth 2 -type f -iname "README*" | head -1`
3. Score each README:
   - Existence: 1 if file exists and is readable, else 0.
   - Length: 1 if word count > 50 (`wc -w`), else 0.
   - Structure: 1 if contains at least one Markdown header (`##`), else 0.
   - Purpose: 1 if clearly states project purpose (subjective but clear if missing).
   - Examples: 1 if includes code/usage example.
   Total score = sum (0-5).
4. Sort by score ascending, pick top 10.
5. Document each with repo path, score, and specific missing elements.

**Implementation:**
```bash
#!/bin/bash
# Scan SuperInstance repos for README quality
echo "Scanning for SuperInstance repos..."
repo_paths=$(find /home -type d -name ".git" -path "*SuperInstance*" 2>/dev/null | sed 's/\/.git//' | sort -u)

declare -A scores
declare -A reasons

for repo in $repo_paths; do
    readme=$(find "$repo" -maxdepth 2 -type f \( -iname "README*" -o -iname "readme*" \) | head -1)
    score=0
    reason=""
    
    if [[ -z "$readme" || ! -f "$readme" ]]; then
        score=0
        reason="No README file found."
    else
        # Existence
        score=$((score + 1))
        # Length
        word_count=$(wc -w < "$readme" 2>/dev/null || echo 0)
        if [[ $word_count -gt 50 ]]; then
            score=$((score + 1))
        else
            reason="${reason} Short (<50 words)."
        fi
        # Structure
        if grep -q "##" "$readme" 2>/dev/null; then
            score=$((score + 1))
        else
            reason="${reason} No Markdown sections."
        fi
        # Purpose
        if grep -qi -e "purpose" -e "what is" -e "overview" "$readme" 2>/dev/null; then
            score=$((score + 1))
        else
            reason="${reason} No clear purpose statement."
        fi
        # Examples
        if grep -q -e "```" -e "example" -e "usage" -e "install" "$readme" 2>/dev/null; then
            score=$((score + 1))
        else
            reason="${reason} No usage examples."
        fi
    fi
    
    scores["$repo"]=$score
    reasons["$repo"]=$(echo "$reason" | sed 's/^ //')
done

echo "Top 10 worst READMEs in SuperInstance repos:"
echo "--------------------------------------------"
for repo in $(echo "${!scores[@]}" | tr ' ' '\n' | sort -k1); do
    echo "$repo | Score: ${scores[$repo]}/5 | Deficiencies: ${reasons[$repo]}"
done | sort -t '|' -k2,2n | head -10
```
