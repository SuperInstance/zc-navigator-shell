### 2. Locate and Score READMEs
For each repo directory, I'll check for a README file and apply the criteria.

**Criteria Scoring (0-5, lower is worse):**
- C1: File exists (1 point)
- C2: Non-zero size (1 point)
- C3: Not placeholder text (e.g., contains more than 3 lines of substantive content) (1 point)
- C4: Includes project title and description (1 point)
- C5: Contains at least one link or code snippet (1 point)

I'll write a simple script to evaluate each.

```bash
for repo in fleet-knowledge plato-torch holodeck-rust fleet-simulator constraint-theory-core cudaclaw flux-runtime; do
    if [ -d "$repo" ]; then
        README=$(find "$repo" -maxdepth 1 -type f -iname "readme*" | head -1)
        SCORE=0
        NOTES=""
        if [ -n "$README" ]; then
            SCORE=$((SCORE + 1))  # Exists
            SIZE=$(stat -f%z "$README" 2>/dev/null || stat -c%s "$README" 2>/dev/null)
            if [ "$SIZE" -gt 0 ]; then
                SCORE=$((SCORE + 1))  # Non-zero size
                CONTENT=$(cat "$README" | head -20)
                LINECOUNT=$(echo "$CONTENT" | wc -l | tr -d ' ')
                if [ "$LINECOUNT" -gt 3 ]; then
                    SCORE=$((SCORE + 1))  # Not placeholder
                else
                    NOTES="$NOTES; minimal lines"
                fi
                if echo "$CONTENT" | grep -q -E "^# .+|^[A-Za-z0-9]" ; then
                    SCORE=$((SCORE + 1))  # Likely has title/description
                else
                    NOTES="$NOTES; missing title/desc"
                fi
                if echo "$CONTENT" | grep -q -E "\[.+\]\(.+\)|http://|https://|`|```" ; then
                    SCORE=$((SCORE + 1))  # Has link or code
                else
                    NOTES="$NOTES; no links/code"
                fi
            else
                NOTES="$NOTES; zero-size file"
            fi
        else
            NOTES="no README found"
        fi
        echo "$repo: Score $SCORE/5 $NOTES"
    fi
done
```
