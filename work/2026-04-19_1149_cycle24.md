### Step 1 — Map the negative space (P0)
**Rocks:**
- Don't assume all repos are under `/home` — some may be in `/mnt`, `/opt`, or other mount points.
- Don't rely on file extensions alone — READMEs could be `.md`, `.txt`, `.rst`, or have no extension.
- Don't evaluate based on subjective "quality" — need objective criteria.
- Don't process binary files or non-text files.
- Don't include repos that are intentionally minimal (e.g., placeholder repos).

### Step 2 — Identify safe channels (P1)
**Safe Paths:**
1. Use `find` command to locate all Git repositories by searching for `.git` directories.
2. Filter to SuperInstance repos (likely under `/home/ubuntu` or `/mnt` based on fleet structure).
3. For each repo, check for README file existence and size.
4. Apply objective scoring criteria:
   - File exists? (0 or 1)
   - File size > 100 bytes? (0 or 1)
   - Contains at least one heading? (0 or 1)
   - Contains installation instructions? (0 or 1)
   - Contains usage example? (0 or 1)
   - Score = sum of criteria (0-5).
5. Sort by score ascending, take top 10.

### Step 3 — Optimize within channel (P2)
**Execution:**
```bash
#!/bin/bash
# Find all .git directories in likely SuperInstance paths
find /home /mnt /opt -type d -name ".git" 2>/dev/null | while read gitdir; do
    repopath=$(dirname "$gitdir")
    # Check for README files (case-insensitive, common extensions)
    readmefile=$(find "$repopath" -maxdepth 1 -type f \( -iname "README*" -o -iname "readme*" \) 2>/dev/null | head -1)
    
    if [[ -n "$readmefile" ]]; then
        # Score the README
        score=0
        [[ -f "$readmefile" ]] && score=$((score+1))
        [[ $(wc -c < "$readmefile" 2>/dev/null) -gt 100 ]] && score=$((score+1))
        grep -q "^#" "$readmefile" 2>/dev/null && score=$((score+1))
        grep -qi "install\|setup\|build" "$readmefile" 2>/dev/null && score=$((score+1))
        grep -qi "example\|usage\|demo" "$readmefile" 2>/dev/null && score=$((score+1))
        
        echo "$score|$repopath|$readmefile"
    else
        # No README found
        echo "0|$repopath|NO_README"
    fi
done | sort -t'|' -k1,1n | head -10
```
