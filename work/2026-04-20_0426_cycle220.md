**2. Criteria for "Needs Improvement"**
A README is flagged as needing improvement if it likely exhibits one or more of these traits:
*   **Missing:** No README file exists.
*   **Skeleton:** Contains only placeholder text (e.g., "# Project Name", "TODO").
*   **Outdated:** Describes goals or APIs that no longer match the current codebase.
*   **Non-Functional:** Lacks basic sections: "Purpose", "Installation", "Usage".
*   **Orphaned:** Belongs to a repository that appears abandoned or experimental.
