### 3. Score each README
**Scoring heuristic (deficiency score, higher = worse):**
- Size < 500 bytes: +3
- Lines < 10: +2
- Missing "Description" or "## Description": +2
- Missing "Installation" or "## Installation": +2
- Missing "Usage" or "## Usage": +2
- Contains "TODO" or "FIXME": +1 per instance
- Contains "Coming soon" or "TBD": +1

**Scoring script (run manually):**
```bash
for readme in $(find /home/cocapn/SuperInstance -type f -iname "README*"); do
    size=$(stat -c%s "$readme" 2>/dev/null || echo 0)
    lines=$(wc -l < "$readme" 2>/dev/null || echo 0)
    content=$(cat "$readme" 2>/dev/null || echo "")
    
    score=0
    if [ $size -lt 500 ]; then score=$((score + 3)); fi
    if [ $lines -lt 10 ]; then score=$((score + 2)); fi
    if ! echo "$content" | grep -qi "## Description\|# Description"; then score=$((score + 2)); fi
    if ! echo "$content" | grep -qi "## Installation\|# Installation"; then score=$((score + 2)); fi
    if ! echo "$content" | grep -qi "## Usage\|# Usage"; then score=$((score + 2)); fi
    todo_count=$(echo "$content" | grep -ci "TODO\|FIXME")
    score=$((score + todo_count))
    tbd_count=$(echo "$content" | grep -ci "Coming soon\|TBD")
    score=$((score + tbd_count))
    
    echo "$score:$readme:$size:$lines"
done | sort -nr | head -10
```
