# Cycle 128 — Navigator
**Task:** Scan SuperInstance repos for READMEs that need improvement. List the 10 worst ones.

---

### Step 1 — Map the negative space (P0)
**Rocks:**
- Don't assume all repos are under `/home` — some may be in `/mnt`, `/opt`, or other mount points.
- Don't rely on file extensions alone; READMEs may be `.md`, `.txt`, `.rst`, or have no extension.
- Don't evaluate repos I cannot access due to permissions.
- Don't waste cycles scanning binary files or non-text content.
- Don't assume the worst READMEs are simply empty; poor structure, outdated info, or missing critical sections are worse.

### Step 2 — Identify safe channels (P1)
**Safe channels:**
- Use `find` command with appropriate paths and filters for text files.
- Limit search to likely directories where fleet repos reside (e.g., `/home`, `/mnt`, `/opt`).
- Use `file` command or `head` checks to confirm text content.
- Apply a scoring heuristic: check for presence of key sections (Overview, Installation, Usage, API), length, and freshness.
- Focus on repos under known fleet orgs (Cocapn, SuperInstance).

### Step 3 — Optimize within channel (P2)
**Execution plan:**
1. Search for README files in likely repo directories.
2. For each, compute a quick quality score based on:
   - File size (bytes) — too small (< 200 bytes) suggests placeholder.
   - Presence of key headers (e.g., `#`, `##`).
   - Date mentions (e.g., "2023", "2024") to gauge freshness.
   - Presence of installation/usage examples.
3. Rank by lowest score, pick top 10 candidates.
4. Output list with repo path and brief reason.

---

**Action:**
```bash
# Search for README files in likely repo locations
find /home /mnt /opt -type f -iname "README*" 2>/dev/null | head -100 > /tmp/readme_list.txt

# Quick scoring script (simplified)
while read -r f; do
    size=$(stat -c%s "$f" 2>/dev/null || echo 0)
    headers=$(grep -c "^#" "$f" 2>/dev/null || echo 0)
    has_install=$(grep -i "install\|setup\|usage" "$f" 2>/dev/null | head -1 | wc -l)
    echo "$size $headers $has_install $f"
done < /tmp/readme_list.txt | sort -n | head -20
```
