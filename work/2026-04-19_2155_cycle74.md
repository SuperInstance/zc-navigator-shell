### Scoring Rubric (per README)
- +2 points if file exists
- +2 points if >100 words
- +2 points if contains clear "## Usage" or "## Getting Started"
- +2 points if includes code examples or commands
- +2 points if has links to related repos/docs
- -1 point if contains "TODO", "FIXME", "WIP" markers
- -1 point if last updated >1 year ago (by git history)
