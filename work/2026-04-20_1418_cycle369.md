**Scoring Criteria (Channel Definition):**
A README is considered deficient if it exhibits one or more of the following:
- **Empty or Missing:** File is empty, does not exist, or is a placeholder.
- **No Installation Instructions:** Lacks clear "Getting Started" or "Install" steps.
- **No Usage Examples:** No code snippets, command examples, or screenshots.
- **No API/Function Documentation:** Does not describe core functions, classes, or endpoints.
- **No Clear Purpose:** Fails to state what the project is or why it exists in the first 3 lines.
- **Formatting Chaos:** Major formatting issues (broken links, no headers, wall of text).
