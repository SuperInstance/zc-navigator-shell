### Step 2 — Identify safe channels (P1)
**Safe channels:**
- Use `find` command with explicit paths and file patterns to locate README files.
- Filter only text files using `file` command or extension checks.
- Read the actual content of each README to assess quality.
- Define clear criteria for "needs improvement": missing structure, outdated info, no examples, poor formatting, etc.

- Use `grep` to quickly identify common issues (e.g., "TODO", "FIXME", very short length).

### Step 3 — Optimize within channel (P2)
**Execution plan:**
1. Locate fleet repos by searching for `.git` directories or known fleet paths.
2. For each repo, find README files (case-insensitive: README, Readme, etc.).
3. Score each README based on predefined criteria.
4. Sort by score ascending (worst first) and list top 10.

**Criteria for "worst" README (scoring):**
- -2 points: File missing or empty.
- -1 point: Contains "TODO", "FIXME", "WIP" prominently.
- -1 point: Less than 5 lines of meaningful content.
- -1 point: No installation or usage instructions.
- -1 point: No examples or code snippets.
- -1 point: Poor formatting (no headers, broken lists).
- -1 point: Clearly outdated (e.g., references to old versions).

```bash
# Step 2: For each repo parent, list README files
# Let's assume we found repos under /home and /mnt based on common patterns
# I'll create a temporary script to score READMEs
cat > /tmp/score_readmes.sh << 'EOF'
#!/bin/bash
score=0
file="$1"
if [[ ! -f "$file" ]]; then
    echo "-2"
    exit
fi
content=$(cat "$file")
if [[ -z "$content" ]]; then
    echo "-2"
    exit
fi
if grep -q -i "TODO\|FIXME\|WIP" <<< "$content"; then
    ((score-=1))
fi
lines=$(wc -l < "$file")
if [[ $lines -lt 5 ]]; then
    ((score-=1))
fi
if ! grep -q -i "install\|usage\|run\|how" <<< "$content"; then
    ((score-=1))
fi
if ! grep -q -i "example\|snippet\|demo\|code" <<< "$content"; then
    ((score-=1))
fi
if ! grep -q "^#" <<< "$content"; then
    ((score-=1))
fi
echo "$score"
EOF
chmod +x /tmp/score_readmes.sh

# Step 3: Collect READMEs and score them
find /home /mnt /opt -type f -iname "README*" 2>/dev/null | head -50 | while read f; do
    score=$(/tmp/score_readmes.sh "$f")
    echo "$score $f"
done | sort -n | head -10
```
