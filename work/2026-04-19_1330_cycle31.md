### 3. README Scoring Rubric
- **Existence (0/1):** README file present?
- **Length (0-3):** 0=empty, 1=<100 chars, 2=100-500, 3=>500.
- **Structure (0-3):** 0=no headers, 1=some headers, 2=sections, 3=standard format (Overview, Install, Use, API).
- **Clarity (0-2):** 0=unclear, 1=understandable, 2=concise & clear.
- **Links (0-2):** 0=no links, 1=internal links, 2=internal + external.
- **Examples (0-2):** 0=none, 1=minimal, 2=working code snippet.
- **Total Score:** Sum of above (max 13). Lower = worse.
