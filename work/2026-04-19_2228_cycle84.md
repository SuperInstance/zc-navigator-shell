## Scoring READMEs

**Scoring criteria (objective, simple):**
1. **Existence:** 0 if missing, 1 if present.
2. **Length:** Lines count (more is better). Score = 1 if lines ≥ 50, else lines/50.
3. **Structure:** Has at least one header (`#`), one list (`-` or `*`), and one code block (```). Score = 1 if all three, else 0.
4. **Links:** Contains at least one markdown link `[text](url)`. Score = 1 if yes, 0 if no.
5. **Recency:** File modified within last 90 days? Score = 1 if yes, 0 if no.

**Composite score:** Sum of above (max 5). Lower = worse.
