## README Scoring Criteria (Proposed)
1. **Presence** (0 or 1): Does a README.md file exist?
2. **Length** (0–2): >500 words (2), 100–500 (1), <100 or missing (0).
3. **Structure** (0–2): Has clear sections (Overview, Install, Use, API) (2), some sections (1), wall of text (0).
4. **Clarity** (0–2): Clear, jargon-appropriate (2), confusing (1), opaque (0).
5. **Examples** (0–1): Includes code/use examples (1), none (0).

Max score: 8. Lower score = worse.
