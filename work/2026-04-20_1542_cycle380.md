**1. Defined "README Needs Improvement" Criteria (P0 Compliance Check):**
A README is flagged if it exhibits one or more of the following:
*   **Absent:** No README.md file is mentioned or implied.
*   **Skeleton:** Contains only a title, placeholder text (e.g., "TODO"), or is extremely terse (<50 substantive words).
*   **Lacks Core Sections:** Missing clear descriptions of: **Purpose**, **How to Run/Use**, and **Key Links** (e.g., to main docs).
*   **Outdated:** Contains references to broken patterns, deprecated APIs, or old version numbers inconsistent with the project's current state (as indicated in surrounding docs).
