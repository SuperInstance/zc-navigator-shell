### Step 1 — Map the negative space (P0)
**Rocks:**
- Don't assume all repos are under `/home` — some may be in `/mnt`, `/opt`, or symlinked.
- Don't evaluate repos without READMEs as "worst" — they're missing, not poor.
- Don't rely only on `find` — need to check for README variants (`README.md`, `README.txt`, `README`).
- Don't process binary files or non‑text READMEs.

---

### Step 2 — Identify safe channels (P1)
**Safe path:**
1. Use `find` with multiple root paths (`/home`, `/mnt`, `/opt`).
2. Filter only directories containing `.git`.
3. For each repo, look for common README filenames.
4. If README exists, score it using heuristics (length, structure, clarity).
5. If missing, flag as "no README".
6. Sort by score ascending, pick bottom 10.

### Step 3 — Optimize within channel (P2)
**Implementation:**

```bash
#!/bin/bash
# assess_readmes.sh

# 1. Find all git repos
repo_paths=()
for root in /home /mnt /opt; do
    [ -d "$root" ] && \
    find "$root" -type d -name ".git" 2>/dev/null | \
    grep -v "/.git/" | \
    while read -r gitdir; do
        echo "$(dirname "$gitdir")"
    done
done | sort -u > /tmp/all_repos.txt

# 2. Score each README
score_repo() {
    local repo="$1"
    local score=0
    local readme=""
    
    # Check for README variants
    for f in "README.md" "README.txt" "README"; do
        if [ -f "$repo/$f" ]; then
            readme="$repo/$f"
            break
        fi
    done
    
    if [ -z "$readme" ]; then
        echo "0:NO_README:$repo"
        return
    fi
    
    # Heuristic scoring
    # 1 point if file > 100 bytes
    if [ $(wc -c < "$readme") -gt 100 ]; then
        score=$((score + 1))
    fi
    
    # 1 point if contains section headers (## or ===)
    if grep -q -E "^## |^===|^---" "$readme" 2>/dev/null; then
        score=$((score + 1))
    fi
    
    # 1 point if contains code blocks (```)
    if grep -q '```' "$readme" 2>/dev/null; then
        score=$((score + 1))
    fi
    
    # 1 point if contains links ([text](http))
    if grep -q -E '\[.*\]\(.*\)' "$readme" 2>/dev/null; then
        score=$((score + 1))
    fi
    
    # 1 point if contains installation or usage instructions
    if grep -q -i -E "install|usage|run|example" "$readme" 2>/dev/null; then
        score=$((score + 1))
    fi
    
    echo "$score:$readme:$repo"
}

# 3. Score all repos
while IFS= read -r repo; do
    score_repo "$repo"
done < /tmp/all_repos.txt > /tmp/readme_scores.txt

# 4. Sort by score (lowest first) and pick worst 10
echo "=== WORST 10 READMEs ==="
sort -t':' -k1n /tmp/readme_scores.txt | head -10
```

**Run it:**
```bash
chmod +x assess_readmes.sh
./assess_readmes.sh
```
