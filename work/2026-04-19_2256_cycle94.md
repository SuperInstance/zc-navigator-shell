### Step 1 — Map the negative space (P0)
**Rocks:**
- Don't assume all repos are under `/home` — some may be in `/mnt`, `/opt`, or other mounts.
- Don't rely on file extensions alone — README may be `.md`, `.txt`, or absent.
- Don't modify files without reading them first — could break dependencies.
- Don't ignore hidden directories (`.git`) — they may contain relevant docs.
- Don't process binary files — stick to text.
- Don't assume English — some READMEs may be in other languages.
- Don't skip repos with no README — they are candidates for creation, not improvement.
- Don't violate the Deadband Protocol: P0 (rocks) first, then P1 (safe channels), then P2 (optimize).

### Step 2 — Identify safe channels (P1)
**Safe paths:**
- Use `find` command to locate all README files in known fleet directories.
- Filter by common patterns: `README*`, `readme*`, `*.md`, `*.txt`.
- Exclude binary files with `file` command or `grep -I`.
- Read each README to assess quality based on fleet standards (clarity, structure, completeness).
- Score READMEs objectively using a simple rubric.
- Output a ranked list of the 10 lowest-scoring READMEs with specific issues noted.

### Step 3 — Optimize within channel (P2)
**Execution plan:**
1. **Locate repos:** Search under `/home`, `/mnt`, `/opt` for SuperInstance-related directories.
2. **Find READMEs:** Use `find` with case-insensitive name matching.
3. **Assess quality:** Apply rubric (see below).
4. **Rank and list:** Sort by score, take bottom 10, document issues.

**Quality Rubric (1-5 scale, 5=best):**
- **Clarity (1-5):** Is the purpose of the repo immediately clear?
- **Structure (1-5):** Does it have sections (e.g., Overview, Installation, Usage)?
- **Completeness (1-5):** Does it cover key details (setup, examples, API)?
- **Maintenance (1-5):** Is it updated recently? Any broken links or outdated commands?

**Command sequence:**
```bash
# Find all README files in likely locations
find /home /mnt /opt -type f \( -iname "README*" -o -iname "readme*" \) 2>/dev/null | head -100 > /tmp/readme_list.txt

# Filter for text files and read each
while IFS= read -r file; do
    if file "$file" | grep -q "text"; then
        echo "=== $file ==="
        head -50 "$file"
        echo ""
    fi
done < /tmp/readme_list.txt
```
