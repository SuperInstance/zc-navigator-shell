### 2. Score each README
**Scoring criteria (lower = worse):**
- **0 points:** File missing or empty (<5 lines)
- **1 point:** Placeholder only (TODO, TBD, "coming soon")
- **2 points:** Minimal content but missing key sections
- **3 points:** Has basic structure but poor formatting
- **4 points:** Good README with minor issues
- **5 points:** Excellent README

**Key sections checked:** Description, Installation, Usage, Examples, API.
