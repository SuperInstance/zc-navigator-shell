## Scoring Rubric
Each README scored 0–10 based on:
- **Structure (0–3):** Has clear sections (Overview, Installation, Usage, API, Contributing).
- **Content (0–3):** Provides meaningful explanation, not just placeholder text.
- **Examples (0–2):** Includes code snippets or usage examples.
- **Links (0–2):** Links to related docs, repos, or resources.

Penalties:
- `-2` if file is empty or contains only `# TODO`.
- `-1` if fewer than 50 words.
