## README Scoring Rubric (For Future Use)
When READMEs are available, score 0-10 (10=worst):

- **-2 points** if missing Installation section
- **-2 points** if missing Usage/Examples section
- **-2 points** if missing API/Configuration details
- **-1 point** if no reference to fleet concepts (Deadband, tiles, rooms)
- **-1 point** if unclear project purpose
- **-1 point** if outdated (e.g., >6 months since update)
- **-1 point** if poor formatting (walls of text, no headers)

Example: A README with no Installation, no Examples, and poor formatting scores 2+2+1 = 5/10 (bad).
