**Safe channels:**

**Execution:**

   ```bash
   find /home /mnt -type f -name "README*" -o -name "README.*" 2>/dev/null | head -50
   ```
   *Output used to identify candidate repos.*

2. **Score each README using rubric (0-10, 10=best):**
   - **+2** if it has a clear project overview.
   - **+2** if it includes installation/usage instructions.
   - **+2** if it documents API or key functions.
   - **+2** if it has examples or code snippets.
   - **+1** if it links to related projects or dependencies.
   - **+1** if it shows recent updates (within last year).
   - **-5** if it's empty, "TODO", or just a title.

3. **Rank by lowest score, select top 10 worst.**
