**Execution plan:**
1. Run find command to list READMEs.
2. For each, extract: repo name, README path, first 10 lines, last modified date.
3. Score each README 1-5 (5=best) based on:
   - Has project description (1 point)
   - Has installation/usage instructions (1 point)

   - Is updated within last year (1 point)

4. Sort by score ascending, take top 10.
5. Present list with scores and reasons.
