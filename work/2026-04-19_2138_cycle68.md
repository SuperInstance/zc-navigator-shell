### Step 2 — Identify safe channels (P1)
**Safe channels:**
1. Use `find` with `-type d -name .git` to locate git repos, then check parent directory.
2. For each repo, look for README files (case-insensitive, common extensions).
3. Score each README based on:
   - **Existence** (0 or 1): Does a README file exist?
   - **Length** (0–2): Too short (<50 words) or excessively long (>1000 words) without structure.
   - **Structure** (0–3): Has clear sections (e.g., Overview, Install, Usage, API, Contributing).
   - **Clarity** (0–3): Language is clear, not overly technical or vague.
   - **Freshness** (0–1): Last updated within last year (check git log).
   - **Links** (0–1): Contains links to related docs, issues, or CI.
   - Total score: 0–11, lower = worse.
4. Filter out:
   - Repos with `.archived` file or `ARCHIVED` in README.
   - Forks (check `.git/config` for upstream).
   - Repos under `deprecated/`, `legacy/`, `experimental/` if explicitly marked.
5. Sort by score ascending, pick top 10.
6. Validate each candidate manually to avoid false positives.

**Scoring script outline:**
```python
import os, re, datetime
def score_readme(repo_path):
    # Find README
    readme_path = None
    for f in os.listdir(repo_path):
        if re.match(r'(?i)readme(\.md|\.txt)?$', f):
            readme_path = os.path.join(repo_path, f)
            break
    if not readme_path: return 0, "No README"
    # Read content
    try:
        with open(readme_path, 'r') as f:
            text = f.read()
    except: return 0, "Unreadable"
    # Score
    score = 0
    words = len(text.split())
    if words > 0: score += 1  # Existence
    if 50 <= words <= 1000: score += 2  # Length good
    elif words < 50: score += 1  # Too short
    else: score += 0  # Too long
    # Structure check
    sections = ['overview', 'install', 'usage', 'api', 'contributing', 'license']
    found = sum(1 for s in sections if re.search(r'#+\s*' + s, text, re.I))
    if found >= 3: score += 3
    elif found >= 1: score += 1
    # Clarity: simple heuristic — sentence length variance
    sentences = re.split(r'[.!?]+', text)
    avg_len = sum(len(s.split()) for s in sentences)/max(len(sentences),1)
    if 5 <= avg_len <= 20: score += 3
    elif avg_len < 5 or avg_len > 40: score += 0
    else: score += 1
    # Freshness (git log)
    # Links
    if re.search(r'\[.*?\]\(.*?\)', text) or 'http' in text:
        score += 1
    return score, os.path.basename(readme_path)
```
