### Scoring Criteria (0-10, higher = worse)
1.  **Missing (10)** – No README.md file in repo root.
2.  **Empty (8)** – README exists but is < 50 characters or only has a title.
3.  **No Examples (5)** – Lacks code snippets, CLI examples, or demo.
4.  **No Installation (5)** – No setup steps, requirements, or `pip install` instructions.
5.  **No Purpose (4)** – Doesn't explain *why* the project exists.
6.  **Broken Links (3)** – Contains URLs that return 404 (requires network check).

8.  **Outdated (2)** – References deprecated APIs or old version numbers.

    if [ ! -f "$readme" ]; then score=$((score + 10)); fi
    if [ -f "$readme" ] && [ $(wc -c < "$readme") -lt 50 ]; then score=$((score + 8)); fi

done | sort -t, -k2 -nr | head -10
