# Cycle 106 — Navigator
**Task:** Scan SuperInstance repos for READMEs that need improvement. List the 10 worst ones.

---

### Step 1 — Map the negative space (P0)
**Rocks:**
- Don't assume all repos are under `/home` — some may be in `/mnt`, `/opt`, or other mount points.
- Don't rely on a single directory scan; need to check multiple known fleet code locations.
- Don't evaluate READMEs without reading their actual content — a short file might be complete for its purpose.
- Don't ignore hidden directories (`.git`) or non-standard README names (`README.txt`, `Readme.md`).
- Don't process binary files or non-text files as READMEs.
- Don't assume the task is only about SuperInstance-branded repos; the fleet includes many related projects.

### Step 2 — Identify safe channels (P1)
**Safe channels:**
- Use `find` command with explicit paths: `/home`, `/mnt`, `/opt`.
- Filter for files named `README*` (case-insensitive).
- Read each file to assess quality based on objective criteria: presence of sections (Description, Usage, Installation), clarity, length, outdated info.
- Limit to first 1000 lines of output to avoid overload.
- Focus on repos under known fleet organizations: `Cocapn`, `SuperInstance`, `plato-*`, `holodeck-*`, `fleet-*`.
- Use `head -n 100` to preview content before full read.

### Step 3 — Optimize within channel (P2)
**Execution plan:**
1. Run a targeted find command across likely directories.
2. For each README, score it based on:
   - Has a project description? (1 point)
   - Has installation/usage instructions? (1 point)
   - Has clear examples or API docs? (1 point)
   - Is longer than 10 lines? (1 point)
   - Mentions fleet concepts (PLATO, deadband, tiles)? (1 point)
   - Total score 0-5.
3. Sort by lowest score, pick top 10.
4. Verify each candidate by reading full content to avoid false positives (e.g., placeholder READMEs that are intentionally minimal).

**Command executed:**
```bash
find /home /mnt /opt -type f -iname "README*" 2>/dev/null | head -n 1000 | while read f; do
  if grep -q "Cocapn\|SuperInstance\|plato-\|holodeck-\|fleet-" "$f" 2>/dev/null; then
    score=0
    [[ $(head -n 5 "$f" | grep -i "description\|summary" | wc -l) -gt 0 ]] && ((score++))
    [[ $(grep -i "install\|usage\|run\|example" "$f" | wc -l) -gt 0 ]] && ((score++))
    [[ $(grep -i "api\|function\|method" "$f" | wc -l) -gt 0 ]] && ((score++))
    [[ $(wc -l < "$f") -gt 10 ]] && ((score++))
    [[ $(grep -i "plato\|deadband\|tile\|fleet" "$f" | wc -l) -gt 0 ]] && ((score++))
    echo "$score:$f"
  fi
done | sort -n | head -20
```
