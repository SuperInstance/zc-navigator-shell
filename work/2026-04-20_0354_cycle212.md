## Execution

### 1. Define Evaluation Criteria
A README "needs improvement" if it exhibits one or more of the following:
- **Missing:** No README file present.
- **Empty:** README exists but is empty or contains only placeholder text (e.g., "TODO").
- **Sparse:** Extremely short (<50 words) with no useful information.
- **Outdated:** Contains references to deprecated APIs, broken links, or old version numbers.
- **Unclear:** Lacks basic sections (Purpose, Installation, Usage, API).
- **Broken Formatting:** Renders poorly on GitHub/GitLab.
