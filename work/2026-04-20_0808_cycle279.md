## README Quality Scoring Rubric
**Criteria (1 point each, max 5):**
1. **Existence:** File is present and named README.md (or .txt).
2. **Structure:** Contains clear sections (e.g., Description, Usage, Installation).
3. **Clarity:** Language is clear, concise, and free of excessive jargon.
4. **Completeness:** Provides enough information to understand the project's purpose and basic use.
5. **Links/References:** Includes links to related repos, documentation, or resources.

Score: 5 = Excellent, 0 = Missing.
